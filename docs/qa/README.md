## How to reproduce the evidence (the battery)

All evidence below is reproducible from a clean checkout. Run the full battery:

```bash
node scripts/qa-verify.mjs        # the all-green gate (writes automated-verification-latest.md)
```

That orchestrates the individual checks; you can also run them piecewise:

```bash
npm run lint                              # ESLint — 0 warnings
npm run typecheck                         # tsc --noEmit (TS strict)
npm run test:run                          # Vitest unit/component — 585 tests, 52 files
npm run test:integration                  # cross-slice business flow — 21 tests
npm run build                             # next build — / stays static; api/* dynamic
npx openspec validate --all --strict      # 9 specs pass
node scripts/check-traceability.mjs       # 33 MVP FRs → spec/plan/test trace
node scripts/check-trajectory.mjs         # per-slice scope audit
node scripts/check-recordings.mjs         # recording manifest check (browser-gated)
node scripts/check-coverage-ratchet.mjs   # guards quality/coverage-baseline.json (95.78% lines)
node scripts/check-eval-ratchet.mjs       # guards the committed eval score (Phase 6)
```

Latest battery result: **Overall Pass** — see
[automated-verification-latest.md](automated-verification-latest.md)
(run 2026-06-27 09:23–09:24 Z).

## What is environment-gated (honest scope)

Two evidence classes are not produced in this environment and are reported
**pending**, never faked:

- **Browser E2E recordings** (chrome-devtools MCP): the recording column of the
  traceability matrix is empty by design — no Playwright (TC-STACK-05), and the
  chrome-devtools MCP is not connected here (ADR-0004). Recordings, a live axe
  scan, and vision-verify are env-gated in Phase 6. JSDOM component tests cover
  rendering in the meantime.
- **Deploy-gated NFRs**: Lighthouse Performance/Accessibility (NFR-PERF-02,
  NFR-A11Y-01 Lighthouse score) and p95 TTFB (NFR-PERF-01) require the live
  Vercel URL and are measured at G7. The locally-verifiable NFRs (AA contrast,
  console-silent, bundle ≤ 200 KB via lazy chunks, DX battery) are verified now.

See `risk-register.md` and the acceptance report's pending section for the full
list and the ops actions required to close them.
